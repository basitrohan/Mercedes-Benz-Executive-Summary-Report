# Mercedes-Benz Executive Summary Report | 2023

A high-level executive dashboard analyzing the 2023 fiscal year performance for Mercedes-Benz. This report focuses on year-over-year (YoY) comparisons, weekly trends, and model-specific performance metrics to drive data-informed strategic decisions.

## 📊 Business Key Performance Indicators (KPIs)

The dashboard tracks four primary financial and operational metrics compared to the Previous Year (Py):

* **Total Profit:** $213M (▼ 2.86% vs Py)
* **Total Revenue:** $1,276M (▼ 3.40% vs Py)
* **Total Units Sold:** 14K (▼ 4.23% vs Py)
* **Profit Margin:** 125.4K (▼ 0.84% vs Py)

## 📈 Key Visualizations

1.  **Profit & Revenue Weekly Trend:** A granular look at performance throughout the year, highlighting peaks (Highest Month) and troughs (Lowest Month) in both profit and revenue streams.
2.  **Model Comparison:** A multi-layered chart comparing sales performance and engineering benchmarks across the vehicle lineup, including the **AMG GT, G-Class, S-Class, EQS, GLE, E-Class, and C-Class**.
3.  **Satisfaction Score:** A qualitative analysis of customer feedback by model, identifying the **GLE** as a top performer with a satisfaction score of **4.1**.
4.  **YoY Performance Sparklines:** Dynamic sparklines located within the KPI cards to visualize the month-by-month trajectory from January to December.


## 🤖 Data Generation
The data used to build this dashboard is **fully synthetic and AI-generated**. 
* **Methodology:** Generated via a custom Python script using libraries such as `Pandas` and `NumPy`.
* **Purpose:** This approach ensures a realistic dataset for demonstrating data visualization and dashboard architectural skills while maintaining data privacy and security.
